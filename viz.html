The Shape of Meaning

Text has measurable structure in embedding space. The shape of how meaning unfolds over time — which semantic modes follow which, at what rhythm — is a compact fingerprint that distinguishes genres, detects style shifts, and carries meaning without any words.

1. Attractor Trajectories

Each sentence is a point in 384-dimensional embedding space. Projected to 2D via the top eigenvectors of the wavelet correlation matrix, the path reveals how each text moves through semantic space. Color = which attractor basin the text occupies.
What you're seeing

O'Reilly oscillates between two regions — definition and example, back and forth. Yeats cycles through a handful of modes with periodic returns. Gatsby traverses everything — maximum entropy, every basin visited, no repetition. These are three fundamentally different ways of unfolding meaning, visible as shapes before you read a single word.

2. Dynamical Smoothness Over Time

How similar is each sentence's attractor state to the next? High = staying in the same semantic mode (expository). Low = jumping between modes (confrontation). The rhythm of this line IS the style.
The key finding

O'Reilly's smoothness never drops — it stays in the same attractor basin at every scale. Gatsby swings wildly between smooth (backstory, exposition) and rough (confrontation, action). The passages where Gatsby's smoothness matches O'Reilly's level are exactly where Fitzgerald writes like a textbook. The passages where it goes negative are the dramatic confrontations. Style is the shape of this line.

3. Transition Graphs

Each node is an attractor basin. Edges show transition probabilities between basins. Thicker/brighter edges = more frequent transitions. The topology of this graph is the text's structural fingerprint.
Shape as meaning

O'Reilly is a barbell — two dominant basins connected by a thick edge. The text bounces between them. Yeats is a ring — cyclic pathways with preferred transitions. Gatsby is a web — every basin connects to every other. You can tell the genre from the graph topology alone, without reading any text. The shape carries the meaning.

4. Semantic Domain Activation

The 384 embedding dimensions cluster into functional groups. These groups fire for interpretable semantic properties — exposition, interiority, formal structure — regardless of topic. The activation pattern across these clusters IS the genre signature.
Semantic domains without words

Cluster "Exposition" fires 6x stronger for technical text than for novels. "Interiority" fires 3x stronger for literary text. "Formal Structure" fires for BOTH technical prose AND poetry — it detects formalism itself. These are not topic labels. They are structural modes of meaning-making, discovered purely from the mathematics of how embedding dimensions co-activate across wavelet scales.

5. The 2.8KB Fingerprint

The entire dynamical signature of a text — visit distribution, transition matrix, autocorrelation profile — fits in 706 numbers. 2,824 bytes. Stable within a work (Gatsby first-half vs second-half: cosine similarity 0.999). Discriminative between works (30x separation ratio).
What this means

A book's entire style of meaning-unfolding compresses to 2.8 kilobytes. Not what it says — how it moves. You could select few-shot examples by matching this fingerprint, steering an AI to write in any dynamical style about any topic. Content and dynamics are orthogonal axes. The shape is the style. The style is the shape.
