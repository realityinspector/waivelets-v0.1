How fast is it, honestly?

Two steps: embed sentences with MiniLM (the only neural network), then classify with a matrix multiply. Here are the real numbers for each step.

Full transparency on the pipeline

The fingerprint has two phases. Phase 1: Embedding — MiniLM (22M params) embeds your sentences into 384-dim vectors. This costs ~5–50ms on CPU depending on text length, or ~140ms for 500 sentences on GPU. Phase 2: Classification — a matrix multiply + 7 scalar statistics. This is the 68μs part. For pre-embedded corpora (search indexes, cached embeddings), you skip Phase 1 entirely and get 10,000+ docs/sec. For raw text end-to-end, the honest number is ~5–50ms on CPU.

Where Waivelets actually wins

The real story isn't "faster than GPT-4" — that's apples to oranges. It's that the entire classifier fits in 38KB. The MiniLM embedding model is 22M params, but that's a one-time cost shared across every downstream task. Once you have embeddings (which many pipelines already do), structural classification is essentially free: a matrix multiply that runs on a microcontroller. The interesting question isn't speed — it's that text has measurable structural dynamics at all, and they're capturable in 7 numbers.

Modes aren't genres

Shakespeare and Darwin share a mode (dialectical) because they both do rapid systematic contrast. Plato and Austen share a mode (discursive) because they're both doing sustained shaped discourse. The modes describe how meaning moves, not what it's about.

The surprising cross-cuts

Drama is 100% dialectical. All 9 plays in our corpus — Shakespeare, Wilde, every one — register as dialectical. Dialogue IS register oscillation. Meanwhile, religious texts cluster convergent: the Gita, Augustine's Confessions, and the KJV Bible all converge to few semantic basins and cycle there. Darwin's two books split across modes: Origin (dialectical, arguing evidence) vs Descent (contemplative, reflecting on implications). Same author, different dynamics.

Three steps, no magic

Embed. Project. Measure. The entire classifier is a matrix multiply.

1
Embed
Each sentence → 384-dim vector via MiniLM (22M params). The only neural network step.
2
Project
Multiply by a 384×26 eigenbasis (37KB). Each sentence lands in a 26-dim attractor space.
3
Measure
Smoothness, oscillation, 4 semantic activations, basin entropy. 7 numbers, 28 bytes. Done.
Where the eigenbasis came from

Continuous wavelet transforms on embedding sequences, building 384×384 correlation matrices across dimensions, extracting topological invariants. That's the expensive telescope. The eigenbasis is the map it produced. You don't need the telescope once you have the map. The compression ratio is 22 million to 1.

What you can build with this

A 28-byte structural fingerprint that runs at 10K docs/sec opens doors that LLMs can't.

AI text detection
If LLM text has different attractor dynamics than human text, the fingerprint detects it at near-zero cost. No API calls, no rate limits.
Style transfer by dynamics
Select few-shot examples by structural fingerprint instead of topic. Two-axis control: WHAT to say × HOW to unfold it.
Persona fingerprints
Each author has a mode distribution. Different LLM personas would too. The deviation from expected is a measure of authenticity.
Edge classification
The 38KB classifier runs anywhere. For pipelines that already embed text, structural classification adds negligible overhead.

Today we took Waivelets from a research notebook to a live, interactive research tool. Starting from the original repo — which contained the raw wavelet analysis code, a 49-text validation corpus, and a static Shiny app — we rebuilt the entire pipeline: wrote fastprint.py (a zero-dependency fingerprinter that classifies text in 68μs), scaled the Gutenberg corpus from 49 to 100 texts across 8 genres and 2,500 years of writing, built a FastAPI server with a live fingerprinting API, designed this interactive frontend with D3.js visualizations, and deployed everything to Railway with auto-deploy from GitHub. The four dynamical modes — convergent, contemplative, discursive, dialectical — held up at scale: drama went 100% dialectical, religious texts clustered convergent, and Darwin's two books split across modes exactly as the theory predicts (Origin = dialectical, Descent = contemplative).
